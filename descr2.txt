1) подключаем все необходимые библиотеки (см readme)
2) проводим анализ сайта (исследуем html код,
    смотрим на параметры авторизации итд)
3) пытаемся авторизоваться на сайте (при этом смотрим, что происходит
    видим, что при нажатии ктопки "вход" на специальный url
    посылается POST запрос. запоминаем параметры POST запроса)
3) запускаем сессию на сайте (см. "google session")
4) делаем GET запрос на страницу сайта с авторизацией
5) из полученного ответа GET запроса достаём нужный нам контент,
    который будет использоваться для авторизации
    (достаём CSRF токен для успешной авторизации)
6) формируем параметры POST запроса
7) делаем POST запрос
8) из полученного ответа достаём html код
9) достаём всю нужную нам информацию с помощью CSS селекторов
10) выводим её в консоль
