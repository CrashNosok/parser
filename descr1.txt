1) подключаем все необходимые библиотека (см readme)
2) проводим анализ сайта (исследуем html код,
    считаем сколько страниц нужно спарсить)
3) генерируем массив с ответами на GET запросы к сайту по ссылкам
    (страницы изменяются с помощью номера в конце ссылки)
4) идём циклом по ответам массиву с ответами с сайта
5) внутри цикла берём html код каждого ответа
6) из html вырезаем нужную нам информацию с помощью CSS селекторов
7) выводим всю информацию в консоль
